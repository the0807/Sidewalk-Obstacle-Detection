## How to use
1. Preset before xml to txt
     ``` shell
     bash xmltotxt_preset.bash
     ```
  
2. xml to txt
     ``` shell
     python3 xmltotxt.py
     ```

3. Preset before split
     ``` shell
     bash split_preset.bash
     ```

4. Split data
     ``` shell
     python3 split_data.py
     ```

3. Preset before train
     ``` shell
     bash train_preset.bash
     ```
